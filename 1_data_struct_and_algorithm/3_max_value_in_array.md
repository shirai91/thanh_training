– Viết hàm getMax(array) tìm giá trị lớn nhất của 1 mảng

template của hàm

```javascript
function getMax(array) {
  // code ở đây
}
```

```javascript
const array1 = [1, 8, 4, 2];
console.log(getMax(array1)); // in ra 8

const array2 = [33, 2, 1, 55];
console.log(getMax(array2)); // in ra 55
```
